## running locally

To run the game you only have to to do following:

```bash
curl https://cdn.intra.42.fr/document/document/30831/minilibx-linux.tgz -o mlx_linux.tgz
tar xpf mlx_linux.tgz
mv minilibx-linux mlx_linux
rm mlx_linux.tgz
cd mlx_linux
make
cd ..
make
./so_long
```
